# Projet Steam's videogames platform

## Description du Projet

Ce projet est réalisé pour le compte d'Ubisoft, une société d'édition de jeux vidéo.
Cette compagnie souhaite étudier le marché de la plateforme de jeux Steam (plateforme de distribution de jeux en ligne) afin de faire ressortir les tendances du secteur.

## Objectifs

* Analyse macro: Étude des éditeurs dominants, des meilleurs jeux, ...
* Analyse par genres: Identification des genres les plus représentés, des plus lucratifs, ...
* Analyse des plateformes: Étude de la disponibilité des jeux sur Windows, Mac et Linux

## Données

steam_game_output.json: Dataset des jeux video de Steam
55 691 lignes, 2 colonnes

## Structure du projet

* ```data/```: Jeux de données
* ```notebook/```: Notebooks Jypiter
